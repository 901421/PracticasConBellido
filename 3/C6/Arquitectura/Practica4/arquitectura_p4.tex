\section{Introducción}
El presente documento detalla la arquitectura del sistema de gestión de pedidos e inventario desarrollado para la Práctica 4. El sistema evoluciona desde una solución monolítica hacia un ecosistema de microservicios orquestados. Se integran patrones de comunicación asíncrona (MOM), seguridad perimetral y el patrón de consistencia eventual \textbf{Saga} para garantizar la integridad de los datos en un entorno distribuido.
